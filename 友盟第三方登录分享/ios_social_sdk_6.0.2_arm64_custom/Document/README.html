更新说明：

v6.0.2

  • 修复SMS和Email的分享不了图片的问题。
  • 修正了获得用户信息的时候,返回授权的问题。
  • 增加了swift3.0的demo示例。
  • 这个版本支持CocoPods.

v6.0.1

  • 优化分享面板
  • 分享未安装客户端或客户端版本不支持时会在分享面板隐藏平台
  • 增加授权和分享提示的国际化文本支持
  • 修复获取用户信息重复SSO授权的问题
  • 修复新浪微博授权直接返回App的问题
  • 新增API可隐藏平台下指定类型，如微信收藏。

v6.0.0

  • 全面重构社会化分享SDK，组织更清晰，第三方平台添加更便捷
  • API接口相对5.x版本更加简洁易用，授权、分享和获取用户信息接口独立
  • 核心库较老版本有更小的size
  • 开源UI，分享面板可灵活定制
